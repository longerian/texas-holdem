## 🎯 如何开始

1. 点击"开始游戏"按钮
2. 你是大盲注（10筹码），电脑是小盲注（5筹码）
3. 每个阶段你可以选择：
   - **过牌 (Check)**: 不加注，等待对手行动
   - **跟注 (Call)**: 跟上对手的下注
   - **加注 (Raise)**: 增加下注金额
   - **弃牌 (Fold)**: 放弃当前局
4. 经过4轮下注后摊牌，比较牌型大小

## 📋 牌型大小（从小到大）

1. **高牌** (High Card)
2. **一对** (One Pair)
3. **两对** (Two Pair)
4. **三条** (Three of a Kind)
5. **顺子** (Straight)
6. **同花** (Flush)
7. **葫芦** (Full House)
8. **四条** (Four of a Kind)
9. **同花顺** (Straight Flush)

## 🎲 游戏规则

### 下注轮次
1. **Preflop**: 每人2张底牌
2. **Flop**: 翻开3张公共牌
3. **Turn**: 翻开第4张公共牌
4. **River**: 翻开第5张公共牌
5. **Showdown**: 摊牌比较大小

### 获胜条件
- 其他玩家都弃牌
- 摊牌时牌型最大
